Typography

Typography is probably the most important aspect of a design system. The typographic scale should help to present the contents and the design itself in an efficient and clear way.

Font Line Height
The line height for different font sizes is calculated using factor 1.4. For example, to calculate the line height for 14px we will do the following: 14px * 1.4 = 20px (line-height).
